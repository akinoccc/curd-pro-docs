# FCurd

一个面向 **Vue 3.4+** 的 CRUD 方案（组件 + 组合式能力），目标是把“表格 / 搜索 / 表单 / 行为”这套常见页面搭建变成可配置、可扩展、可适配 UI 框架的基础设施。

## 特性概览

- **声明式字段/列**：基于字段元信息生成搜索/表单/表格
- **可插拔 UI Driver**：核心逻辑与 UI 解耦，支持不同 UI 框架适配
- **可组合**：既可开箱使用，也可拆分组合渲染器自行布局
- **路由搜索同步**：可选将搜索条件与 URL 互相同步（基于 `vue-router`）
- **类型友好**：面向业务实体/查询条件的类型推导
